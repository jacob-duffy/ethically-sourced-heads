## Key Directories

### `frontend/`

Vite + Vanilla JS + WebGL application served on GitHub Pages.

- `frontend/src/main.js` — Bootstrap app, fetch `heads.json` on load
- `frontend/src/viewer.js` — HeadPreviewComponent
- `frontend/public/heads.json` — **Single source of truth** for inventory; generated by `aggregate.py`
- `frontend/public/textures/` — PNG texture files downloaded from `textures.minecraft.net` and named as `{texture_hash}.png`; generated by `aggregate.py`, committed to git

### `scripts/`

Local Python tooling and JSMacros macros for aggregation.

- `scripts/macros/` — JSMacros macro files for exporting head data
  - `scripts/macros/head-tracker.js` — In-game macro that exports heads to `scripts/input/untracked/`
- `scripts/input/` — Head data file directory with two subdirectories:
  - `scripts/input/untracked/` — Drop folder for newly generated JSMacros head data files (`head_data_<timestamp in ms>.json`)
  - `scripts/input/tracked/` — Archive of processed head data files (moved here after aggregation)
- `scripts/aggregate.py` — Main aggregation script; runs locally, reads files from `scripts/input/untracked/`, merges into `frontend/public/heads.json`, moves files to `scripts/input/tracked/`, commits & pushes to GitHub

### `docs/`

Reference documentation for developers and maintainers.

## Build Outputs & Ignore Patterns

| Directory | Purpose | Git-Tracked? |
|-----------|---------|---|
| `scripts/input/untracked/` | New head data files awaiting aggregation | No (git-ignored) |
| `scripts/input/tracked/` | Archive of processed head data files | Yes (git-tracked) |
| `frontend/dist/` | Vite build output (deployed by GitHub Actions) | No (git-ignored) |
| `frontend/node_modules/` | npm packages | No (git-ignored) |
| `frontend/public/heads.json` | Generated inventory (source of truth) | **Yes** |
| `frontend/public/textures/*.png` | Downloaded textures | **Yes** |

## Environment Files

No `.env` files required. No secrets.

## Git Repository Structure

### Branches

- `main` — Production branch, always deployed to GitHub Pages
- Feature branches (e.g. `feature/webgl-viewer`, `feature/search-ui`) — Merge to `main` when ready

### Commits

- Manual commits: Feature development, bugfixes, documentation
- Auto-commits: Each aggregation script run produces `chore: update inventory` commit

## Quick Local Setup

```bash
# Clone the repo
git clone <repo-url>
cd civmc-headshop

# Frontend dev server
cd frontend
npm install
npm run dev                     # http://localhost:5173

# Aggregation script (no dependencies needed)
cd ..
python scripts/aggregate.py    # Requires JSMacros files in scripts/input/untracked/
```
