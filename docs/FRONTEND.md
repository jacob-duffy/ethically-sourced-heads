# Frontend Implementation

## Tech Stack

- Vite (build tool, dev server)
- Vanilla JavaScript (no framework)
- CSS3 (styling, dark theme)
- WebGL 1.0 (3D head previews, refactored from existing headViewer.html)

## Features

### Search & Filter

- Real-time name search
- Rarity dropdown
- Multi-select tag filters
- Stock status toggle
- Sort by: name, price, rarity

### Catalog Grid Display

- Cards show head preview (160×160 px WebGL canvas), name, rarity badge, price summary
- On hover: Head rotates clockwise continuously (auto-rotation)
- Card background highlights on hover
- Click card to expand into detail view

### Expanded Detail View (Modal)

- Full-screen or large modal overlay
- Large head preview (400×400 px WebGL canvas)
- Full 360° interactive rotation (mouse drag: left-right for yaw, up-down for pitch)
- Detailed info: full name, rarity tier, complete pricing breakdown (diamonds/emeralds/iron), all tags, description, stock status
- "Back" button to return to catalog grid
- Close button (X) to dismiss modal
- Optional: Share button to copy URL to this specific head

### Pagination

- Display 20-30 heads per page
- Previous/Next buttons + page indicator
- 1,280 heads = ~43-64 pages

### Responsive Design

- Mobile-friendly grid layout (cards stack on small screens)
- Touch-friendly buttons and inputs
- Expanded modal adapts to mobile (full-screen on small screens)
- Accessible keyboard navigation (arrow keys for pagination, ESC to close modal)

### State Management

- URL query params (search, rarity, page) for bookmarkable catalog states
- Direct links to expanded head views (e.g. `/shop/head/:id`)
- Users can share filtered views or link directly to specific heads

### WebGL Rendering & Performance

- Lazy load viewers only for visible heads in catalog grid (Intersection Observer)
- Unload off-screen viewers to free GPU memory (maintain 3-5 active canvases max)
- Expanded modal viewer runs independently (separate WebGL context)
- Fallback to static texture image if WebGL fails
- Continuous animation loop for hover rotation; interactive mouse events only in expanded view

## WebGL Head Previews

### Catalog Grid Mode

- Fixed front-facing angle (non-rotatable by default)
- On card hover: clockwise auto-rotation (~30°/sec)

### Expanded View Mode

- Full 360° interactive rotation via mouse drag
- Left-right drag controls yaw (horizontal spin)
- Up-down drag controls pitch (vertical tilt, clamped)

## Deployment

- Hosted on GitHub Pages (free)
- Auto-deploys on every `git push` to `main` via a GitHub Actions workflow that runs `npm run build`
- No environment variables required (all data is static)
- `heads.json` and textures served via GitHub Pages CDN with long-lived cache headers
- HTTPS provided automatically (required for WebGL cross-origin texture loading)
