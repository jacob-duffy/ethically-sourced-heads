# Project Overview

## High-Level Details

A public-facing storefront webpage for a custom player head shop on CivMC (a Minecraft Vanilla+ server), operated by a single player-run company. The site serves as a catalog and discovery tool — no on-site purchasing. Players join the shop's Discord server to place orders.

**Expected traffic:** no more than 50 concurrent users.

---

## Inventory

- Heads are primarily custom/decorative player heads with custom textures.
- Each head listing includes a **real-time 3D WebGL preview** (rendered using the skin texture), name, rarity, tags, price, and in/out-of-stock status.
- Stock is shown as available or unavailable only (no quantity counts).
- All inventory, pricing, and tags are managed manually by the shop owner.

---

## Rarity Tiers

| Tier | Description |
| ------ | ------------- |
| Junk | Common, low-value heads |
| Uncommon | Moderately interesting heads |
| Rare | Harder to find or visually distinct heads |
| Legendary | Premium or highly sought-after heads |
| Player | Special tier for heads modeled after notable CivMC players |

---

## Pricing

Currency is Minecraft commodities: **diamonds**, **emeralds**, and **iron**. Prices are set per head by the shop owner.

---

## Search & Filtering

- Search by name
- Sort/filter by rarity tier
- Filter by manually assigned tags (e.g. "animal", "food", "holiday")
- Tag categories are extensible — new tags can be added by the owner over time

---

## Ordering

Players cannot purchase directly through the site. A **Discord invite link** is prominently displayed, directing players to a designated order channel.

---

## Branding

The site is tied to a single player's company on CivMC. Company name and branding are to be determined.
